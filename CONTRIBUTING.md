# 贡献指南

感谢你对 alidns-ddns 的关注！

## 如何贡献

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/my-feature`)
3. 提交更改 (`git commit -m 'feat: add some feature'`)
4. 推送到分支 (`git push origin feature/my-feature`)
5. 创建 Pull Request

## 开发规范

- 遵循 [Conventional Commits](https://www.conventionalcommits.org/) 提交规范
- 确保 `cargo fmt --check` 和 `cargo clippy` 通过
- 新功能请添加相应文档
- 修复 bug 请附带复现步骤

## 报告问题

使用 [GitHub Issues](https://github.com/tongtf/alidns-ddns/issues) 报告问题，请包含：

- 操作系统和 Rust 版本
- 复现步骤
- 预期行为和实际行为
- 相关日志输出

## 安全问题

请参阅 [SECURITY.md](SECURITY.md) 了解安全漏洞报告流程。
